## Requirements
- Node.js 16.x
- TypeScript 4.x
- Parcel 2.x
- @types/node
- ts-node

## Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/SAMPI007able/flipPe.git
   ```


## Usage
To run the application, use the following command:
```bash
   npm start
   ```
To test the application for debugging, use the following command:
```bash
   npm run dev  
```
